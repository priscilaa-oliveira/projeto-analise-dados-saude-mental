# Projeto de Análise de Dados – Trabalho Remoto e Saúde Mental
Projeto de conclusão do curso de Análise de Dados

## Acesso ao Projeto
- [Repositório completo no GitHub](https://github.com/priscilaa-oliveira/projeto-analise-dados-saude-mental)
- [Notebook no Google Colab](https://colab.research.google.com/drive/1MxpPKiXPSgFtsbly6SZHtWP1ADiUXoWz?usp=sharing)

## Objetivo
Este projeto tem como objetivo realizar a análise exploratória e o tratamento dos dados relacionados ao trabalho remoto e saúde mental, preparando as informações para a criação de visualizações claras e interativas.

## Ferramentas utilizadas
- Python (Pandas)
- Jupyter Notebook
- Excel
- Power BI

## Descrição do projeto
A análise aborda aspectos como perfil etário, gênero, rotina de trabalho remoto e impactos na saúde mental, com foco na organização e interpretação dos dados.

## Estrutura do projeto
- Notebook em Python com análise exploratória
- Dataset utilizado (sintético)
- Visualizações e gráficos explicativos
- Relatório resumido das análises
